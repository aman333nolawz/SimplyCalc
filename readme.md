# SimplyCalc - Algebraic Calculus Simplification - C Library

A lightweight, modular C library for numerical calculus operations.  
Implements Grünwald-Letnikov derivatives/integrals for theoretical physics and pure math research.
Also implements topics of the research paper given.

## Features (So far :\ )
- Integer order derivatives
- Integer order antiderivatives

## Quick Start
```bash
git clone https://github.com/CodeFromKerala/SimplyCalc
cd SimplyCalc
make # for local install
sudo make install # for global install
```

## Implementing and Running Scripts
```bash
gcc file_name.c -lsimplycalc -lparsefunc -lmathobj -lm -o output_file_name
```
